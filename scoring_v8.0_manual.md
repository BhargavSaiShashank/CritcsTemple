## 🔨 2. Dynamic Refinements (The Engine)

Protocol V8.0 uses **Non-Linear Refinements** to ensure the punishment matches the failure and the reward matches the greatness.

### 🔴 The Hammer (Gradient Penalties)
Unlike fixed deductions, V8.0 uses a sliding scale.
*   **Narrative Foundation**: Penalizes reviews if the Narrative average is below 6.5. A 6.4 gets a minor deduction, while a 3.0 gets a massive drop.
*   **Execution Foundation**: Penalises if Direction/Acting average is below 6.5.
*   **Consistency Check**: Penalizes "Lopsided" films where the gap between the highest and lowest pillar is greater than 2.0.

### 🌟 The Zenith (Synergy Boosts)
*   **Core Synergy**: If both Narrative and Execution are 9.0+, they provide a compounding boost to the final score.
*   **Legendary Harmony**: If all 5 pillars are 8.5+, a flat +0.10 "All-Rounder" bonus is applied.
