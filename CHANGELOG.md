# Changelog

All notable changes to this project will be documented in this file.

## [1.2.0] - 2026-03-08

### Added
- **Archival Hero Carousel**: Implemented a dynamic hero section that cycles through the latest 5 imprints.
- **Ken Burns Atmosphere**: Added smooth poster scaling and atmospheric cross-fades using `framer-motion`.
- **Verdict Synchronization**: Background and UI accent colors now synchronize with the active cinematic verdict.

### Fixed
- **Mobile Touch Interaction**: Resolved `Unable to preventDefault inside passive event listener` error by attaching listeners directly to the DOM with `{ passive: false }`.
- **Initialization Race Condition**: Fixed `ReferenceError` where `fetchReviews` was called before its initialization in the render cycle.
- **Vertical Overflow (Mobile)**: Corrected layout scaling and padding for small viewports (iPhone SE) to prevent content clipping.

### Changed
- **Mobile Hierarchy**: Reordered Hero content to a "Metadata-First" architecture (Title/Actions above Poster).
- **Navigation Repositioning**: Moved carousel navigation pills below the visual imprint on mobile for better usability.
- **Optimized Render Chain**: Streamlined the data-fetching logic to eliminate redundant re-renders and loading flickers.

### Platform Sync
- **Android Integration**: Certified all V1.2.0 features on the Android native project with a fresh production build and Cap sync.
